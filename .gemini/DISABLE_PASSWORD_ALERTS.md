# How to Disable Chrome Password Breach Alerts

## Method 1: Disable in Chrome Settings

1. Open Chrome and go to: `chrome://settings/passwords`
2. Or navigate via: **Settings** → **Autofill and passwords** → **Google Password Manager**
3. Click the **Settings** gear icon (⚙️) in the Password Manager
4. Toggle **OFF** the option: **"Help you know if your passwords are safe"**

## Method 2: Disable Password Manager Completely

1. Go to: `chrome://settings/passwords`
2. Toggle **OFF**: **"Offer to save passwords"**
3. Toggle **OFF**: **"Auto Sign-in"**

## Method 3: Just Close This Alert

- Click the **"Close"** button on the alert
- The alert will stop showing for this specific password breach

## Important Security Note ⚠️

**This alert is a security feature.** If you disable it, you won't be warned when your passwords are compromised. 

Instead of disabling, consider:
- Click **"Check passwords"** to see which password was breached
- Change the compromised password
- Use unique passwords for each site
- Enable 2-factor authentication where possible

## Quick Fix for Development

If you're seeing this during development because you're using test credentials:

1. Use Chrome's Incognito Mode for testing (Ctrl + Shift + N)
2. Or use a different browser profile for development
3. Or click "Close" and ignore the alert
